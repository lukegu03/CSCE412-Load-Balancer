Detailed Description

Runs a load balancer simulation with streaming and processing servers.

This program simulates a system that generates random requests over time, optionally filters them through a firewall, queues them into one of two load balancers (streaming vs processing), and assigns them to available servers. The system dynamically scales the number of servers based on queue load.
